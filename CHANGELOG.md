# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-10
### Added
- Google Maps integration (Streets, Satellite, Terrain, Hybrid).
- Hydraulic Infrastructure layer (Rivers, Drains, Barrages).
- Live news feed from DayakDaily and Land & Survey Dept.
- PropertyGuru and SarawakProjects integration in Market tab.
- GitHub Actions workflow for automated deployment to GitHub Pages.
- Comprehensive project documentation in `docs/` folder.

### Changed
- Improved land parcel detection accuracy.
- Cleaned up UI by removing environmental layers and property markers.
- Updated LASIS links to official domains.

## [1.0.0] - 2026-04-09
### Added
- Initial release of AquaScope Kuching.
- Basic flood simulation (0-60m).
- Leaflet-based map with OpenStreetMap tiles.
- Mock LASIS land parcel data.
- Basic property market simulation.
